# securAI — Checkout‑Free Smart Store

An interactive 3D pitch deck for **securAI**, a fully autonomous, checkout‑free retail store powered by computer vision, sensor fusion, and **100% solar energy**. Built for the **NURAI 2026** National University Robotics & AI Championship (Community Service track), University of Baghdad.

🔗 **Live:** https://securai-smart-store.vercel.app

## What it is

A presenter‑driven, single‑file interactive experience (no build step, runs in any modern browser). The Team Leader advances scene‑by‑scene with the keyboard; each step plays a 3D animation and a glassmorphism explanation card.

### Controls
- `→` / `Space` / `Enter` — next · `←` / `PageUp` — back
- `R` — replay scene · `F` — fullscreen
- Drag to move the explanation card · drag its corner to resize · double‑click to reset (position is saved per step)
- Live‑demo hotkeys: `W` (Wi‑Fi sensing), `A` (theft alert replay), `S` (financial stress‑test)

## Highlights
- **Customer POV** — first‑person walk‑through with an interactive phone/app, realistic product, and a live basket
- **The AI Core Engine** — cinematic reactor‑core opener for the perception/decision pipeline
- **Wi‑Fi CSI sensing** — cameras sleep until motion is detected in a grid sector (energy saving)
- **100% solar / off‑grid** — animated day↔night showcase proving full autonomy from the utility grid
- **Security / Alert mode** — privacy‑first theft handling: verify, don't accuse
- **Autonomous fulfilment robot + drone delivery** future roadmap
- **Financial model** — 400 m² scenario with a live break‑even chart (≈ month 8)
